About TweetURLApplicationTapzo




TweetURLApplicationTapzo is a twitter application that uses Twitter api to authenticate the user and display the twitter feed to the user. It has a front end interface, where user can enter his twitter id, to access his feed. User can see all the URLs in his twitter and can login at any time.
